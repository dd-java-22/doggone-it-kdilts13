# User Instructions

## Overview

**Doggone It** is a dog breed identification app that allows users to analyze images of dogs, view predicted breeds, and save scans with additional information such as breed facts, notes, and favorites.

The app uses on-device machine learning to generate breed predictions and retrieves additional breed information using an external API.

---

## Getting Started

1. Launch the app.
2. Log in using the login screen.
3. After logging in, you will be taken to the **Home** screen.

---

## Navigation Overview

The app uses a bottom navigation bar with the following options:

* **Home** – Main landing screen
* **Camera** – Capture a new photo using the device camera
* **Gallery** – Select an image from your device
* **Saved Scans** – View previously saved scans
* **Settings** – Adjust app preferences

---

## Analyzing an Image

### Option 1: Take a Photo

1. Tap the **Camera** icon in the bottom navigation.
2. Capture a photo using your device camera.
3. Confirm the photo to proceed to analysis.

### Option 2: Select from Gallery

1. Tap the **Gallery** icon in the bottom navigation.
2. Choose an image from your device.
3. The selected image will be sent to the analysis screen.

---

## Viewing Prediction Results

After selecting or capturing an image:

1. The **Analysis** screen will display a list of predicted dog breeds.
2. Predictions are ranked by confidence.
3. Review the list of predictions.

### Selecting a Prediction

* Tap on a prediction to select it.
* The selected prediction will be used for:

    * saving the scan
    * retrieving breed facts

---

## Saving a Scan

1. After selecting a prediction, choose the option to save the scan.
2. The scan will be stored locally on your device.
3. Saved scans can be accessed later from the **Saved Scans** screen.

---

## Viewing a Saved Scan

1. Open **Saved Scans** from the bottom navigation.
2. Tap on any scan to open the detail view.

### Scan Detail Screen

The scan detail screen includes:

* The original image
* Selected breed prediction
* Breed facts (from the Dog API)

---

## Switching Between Facts and Notes

On the scan detail screen:

* Use the toggle or button to switch between:

    * **Facts view** – Displays breed information from the API
    * **Notes view** – Allows you to enter personal notes

---

## Adding and Editing Notes

1. Switch to the **Notes** view.
2. Enter text in the notes field.
3. Save your changes.

Notes are stored locally and persist between sessions.

---

## Marking a Scan as Favorite

1. Open a scan from **Saved Scans**.
2. Tap the **Favorite** toggle/button.

Favorite scans can be filtered in the gallery.

---

## Using the Saved Scans Gallery

Open **Saved Scans** to view all stored scans.

Features include:

* **Grid layout** of saved images
* **Sorting options** (based on selected settings)
* **Favorites filter** to show only favorite scans

---

## Settings

Open the **Settings** screen from the bottom navigation.

Available options include:

* **Default sort option** for saved scans
* **Sort direction**
* **Column count** for the gallery layout

Changes are saved automatically and applied to the gallery.

---

## Permissions

The app may request access to:

* Camera (for taking photos)
* Media/gallery (for selecting images)

Grant these permissions when prompted to enable full functionality.

---

## Notes and Limitations

* Breed predictions are generated by a machine learning model and may not always be accurate.
* Breed facts require an internet connection and a valid API key configuration.
* The app stores all scan data locally on the device.

---

## Summary

Typical usage flow:

1. Capture or select an image
2. Review predictions
3. Select a breed
4. Save the scan
5. View facts or add notes
6. Manage scans in the gallery

This workflow represents the primary functionality of the application.
